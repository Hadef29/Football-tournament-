How ties are broken
  1. Points — win 3, draw 1, loss 0.
  2. Goal difference — goals for minus goals against, highest first.
  3. Goals scored — most goals for, highest first.
  4. Head-to-head — a mini-table using only matches between the tied teams.
  5. Discipline — fewest red cards first, then fewest yellow cards.
